Configurar API Key (Obligatorio)

Esta web utiliza Inteligencia Artificial avanzada para redactar tu CV. Para que funcione, necesitas conectar una "llave" gratuita de Groq Cloud.

¿Es gratis? SÍ, 100% GRATUITO. No necesitas pagar nada ni meter tarjeta en Groq para el uso personal básico.

  1. Entra en console.groq.com/keys
  2. Inicia sesión (con tu cuenta de Google).
  3. Pulsa en "Create API Key".
  4. Copia el código (empieza por gsk_).
  5. Pégalo en la barra superior de esta web y guarda.
